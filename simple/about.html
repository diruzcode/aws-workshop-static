Acerca de este Workshop

Este workshop está diseñado para ayudarte a comprender cómo desplegar sitios web en AWS utilizando S3 y CloudFront, configurando dominios personalizados con Route 53 y certificados SSL con Certificate Manager.

Aprenderás a:

  • Crear y configurar buckets de S3 para alojar sitios web estáticos
  • Configurar distribuciones de CloudFront para mejorar el rendimiento
  • Gestionar dominios personalizados con Route 53
  • Implementar certificados SSL con Certificate Manager
  • Utilizar CloudFront Functions para manejar rutas dinámicas en aplicaciones React
  • Automatizar despliegues con scripts
